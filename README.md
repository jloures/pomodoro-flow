# Pomodoro Flow

A minimal, customizable Pomodoro timer with ambient sounds generated via the Web Audio API.

## Features

- **Timer**: Work, short break, and long break modes with auto-cycling
- **8 Ambient Sounds**: White noise, pink noise, brown noise, rain, ocean waves, binaural focus, campfire, fan — all generated programmatically
- **Customizable**: Adjust durations, sound, volume, and auto-start behavior
- **Keyboard Shortcuts**: Space (start/pause), R (reset), S (skip)
- **Dark Theme**: Easy on the eyes during focus sessions
- **Responsive**: Works on desktop and mobile
- **Offline-Ready**: Zero external dependencies, everything runs locally

## Usage

Open `index.html` in a browser, or visit the [live site](https://jloures.github.io/pomodoro-flow).
